# StatTierMaker
## General idea:

### First step.
1. User creates a tier list template (or uses default).
2. Tier list template consists of name, description, list of entities.
3. Entity has name, description, parameters.
4. Each parameter has name, value, coefficient.

### Second Step.
1. User selects a template and creates blank tier list.
2. User add entities into the tier list.
3. User saves tier list.
4. User shares tier list as json file.
